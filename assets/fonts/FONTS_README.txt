REQUIRED FONTS FOR VOCAB ZOO THEME

This theme requires two font families to be placed in this directory:

1. FREDOKA (Primary/Heading Font)
   - File: fredoka-bold.ttf (or Fredoka-Bold.ttf)
   - Weights needed: Bold (700)
   - Download from: Google Fonts - https://fonts.google.com/specimen/Fredoka
   - Used for: Titles, buttons, game names, all headings

2. NUNITO (Body/Reading Font)
   - File: nunito-regular.ttf (or Nunito-Regular.ttf)
   - Weights needed: Regular (400)
   - Download from: Google Fonts - https://fonts.google.com/specimen/Nunito
   - Used for: Body text, instructions, descriptions

INSTALLATION:
1. Download both fonts from Google Fonts
2. Place the .ttf files in this directory (/assets/fonts/)
3. Ensure filenames match exactly:
   - fredoka-bold.ttf
   - nunito-regular.ttf
4. The theme file will automatically load them

ALTERNATIVE:
If you prefer different rounded fonts, you can substitute with:
- Primary: Quicksand, Comfortaa, or Baloo
- Body: Open Sans, Lato, or Rubik
(Update the font paths in vocab_zoo_theme.tres if you change fonts)
